# TermEdit

终端内的文本编辑器，但是有可点击的UI!

## 特性

- 完全鼠标驱动
- 不需要学习快捷键
- 标准快捷键支持

## 安装

```bash
pip install -r requirements.txt
```

## 使用

```bash
# 新建文件
python termedit.py

# 打开文件
python termedit.py myfile.txt
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+N | 新建 |
| Ctrl+O | 打开 |
| Ctrl+S | 保存 |
| Ctrl+Q | 退出 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Ctrl+X | 剪切 |
| Ctrl+C | 复制 |
| Ctrl+V | 粘贴 |
| Ctrl+A | 全选 |
| F10    | 打开菜单 |
| ESC    | 关闭菜单 |

## 技术栈

- Python 3.10+
- Textual
